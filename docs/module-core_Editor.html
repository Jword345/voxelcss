    • core/Editor

    Creates all the mouse events necessary to allow the user to add and remove blocks from a World with the added option to autosave all changes.
    Source:
    • Editor.js, line 1
    Examples

    Create an editor and add a voxel

    const Voxelcss = require('voxelcss');
let scene = new Voxelcss.Scene();
let world = new Voxelcss.World(scene);
scene.attach(document.body);
let dimensions = [0, 0, 0, 100];
editor.add(new Voxelcss.Voxel(...dimensions));

    Add a voxel with grass mesh

    let mesh = Voxelcss.meshes.grass;
let grass = new Voxelcss.Voxel(..dimensions, {mesh});
editor.add(grass);
